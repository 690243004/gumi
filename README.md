## Install

```
npm i gumi -S
yarn add gumi
```

## Quickstart

```js
import Vue from 'vue'
import { install } from 'gumi'
import 'gumi/lib/index.css'
install(Vue)

// or
import { Button } from 'gumi'
Vue.component(Button.name, Button)
```

For more information, please refer to Quick Start in our documentation.
